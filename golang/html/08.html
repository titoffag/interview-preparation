Условие задачи

Колода состоит из 52 карт. Каждая карта обозначается одним из тринадцати значений ( 2 \textbf{2} 2, 3 \textbf{3} 3, 4 \textbf{4} 4, 5 \textbf{5} 5, 6 \textbf{6} 6, 7 \textbf{7} 7, 8 \textbf{8} 8, 9 \textbf{9} 9, T \textbf{T} Ten, J \textbf{J} Jack, Q \textbf{Q} Queen, K \textbf{K} King, A \textbf{A} Ace) и одной из четырех мастей ( S \textbf{S} Spades, C \textbf{C} Clubs, D \textbf{D} Diamonds, H \textbf{H} Hearts). Выдуманная игра 3-Покер происходит следующим образом. 1. Изначально все n n n игроков получают по две карты из колоды. 2. После этого на стол выкладывается одна карта из той же колоды. 2. Выигрывают те игроки, у которых собралась самая старшая комбинация. Для определения самой старшей комбинации, которая собралась у i i i-го игрока, используются следующие правила: ∙ \bullet ∙ если две карты у игрока в руке и карта на столе имеют одинаковое значение, игрок собрал комбинацию ‘ Сет   со   значением   x  ’ \lq\textit{Сет со значением $x$}\ \text{\textquoteright} ‘Сет со значением x ’; ∙ \bullet ∙ если из двух карт у игрока в руке и карты на столе можно выбрать две карты с одинаковым значением x x x, игрок собрал комбинацию ‘ Пара   со   значением   x  ’ \lq \textit{Пара со значением $x$}\ \text{\textquoteright} ‘Пара со значением x ’; ∙ \bullet ∙ иначе, берется карта с самым старшим значением из двух карт у игрока в руке и карты на столе, тогда игрок собрал комбинацию ‘ Старшая   карта   x  ’ \lq \textit{Старшая карта $x$} \ \text{\textquoteright} ‘Старшая карта x ’. Любой сет \textit{сет} сет старше пары \textit{пары} пары, а любая пара \textit{пара} пара старше комбинации старшая   карта \textit{старшая карта} старшая карта. Из одинаковых комбинаций старше та, у которой старше значение. Если одинаковая самая старшая комбинация есть у нескольких игроков, все они объявляются выигрывшими. Вы — первый игрок. Вам известно, какие карты получил на руки каждый игрок. Определите, какую карту можно выложить на стол, чтобы вы оказались в числе победителей.

Входные данные

Каждый тест состоит из нескольких наборов входных данных. Первая строка содержит целое число t t t ( 1 ≤ t ≤ 1 0 3 1 \le t \le 10^3 1≤t≤103) — количество наборов входных данных. Далее следует описание наборов входных данных. Первая строка каждого набора входных данных содержит целое число n n n ( 2 ≤ n ≤ 25 2 \le n \le 25 2≤n≤25) — количество игроков. Следующие n n n строк каждого набора входных данных содержат описания двух карт, разделенных пробелом — карты, которые получил на руки i i i-й игрок. Описание карты состоит из двух символов, записанных подряд: значения и масти.

Выходные данные

Для каждого набора входных данных выведите в первой строке количество карт k k k, которые можно выложить на стол для вашей победы. В следующих k k k строках выведите описания этих карт. Выводить описания можно в любом порядке. Разберем первый пример. В первом наборе входных данных для победы первого игрока можно выложить на стол десятку ( T \textbf{T} T), тогда первый игрок выиграет с комбинацией сет со значением T \textbf{T} T. Во втором наборе входных данных невозможно добиться победы первого игрока. В третьем наборе входных данных для победы первого игрока можно выложить на стол семерку ( 7 \textbf{7} 7), тогда первый игрок выиграет с комбинацией пара со значением 7 \textbf{7} 7. В четвертом наборе входных данных невозможно добиться победы первого игрока.
