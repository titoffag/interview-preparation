Условие задачи

В офисе стоит кондиционер, на котором можно установить температуру от 15 15 15 до 30 30 30 градусов. В офис по очереди приходят n n n сотрудников. i i i-й из них желает температуру не больше или не меньше a i a_i ai​. После прихода каждого сотрудника определите, можно ли выставить температуру, которая удовлетворит всех в офисе.

Входные данные

Каждый тест состоит из нескольких наборов входных данных. Первая строка содержит целое число t t t ( 1 ≤ t ≤ 1 0 3 1 \le t \le 10^3 1≤t≤103) — количество наборов входных данных. Далее следует описание наборов входных данных. Первая строка каждого набора содержит целое число n n n ( 1 ≤ n ≤ 1 0 3 1 \le n \le 10^3 1≤n≤103) — количество сотрудников. i i i-я из следующих n n n строк каждого набора входных данных содержит требование к температуре от i i i-го сотрудника: либо ≥ a i \geq a_i ≥ai​, либо ≤ a i \leq a_i ≤ai​ ( 15 ≤ a i ≤ 30 15 \le a_i \le 30 15≤ai​≤30, a i a_i ai​ — целое число). Требование ≥ a i \geq a_i ≥ai​ означает, что i i i-й сотрудник желает температуру не ниже a i a_i ai​; требование ≤ a i \leq a_i ≤ai​ означает, что i i i-й сотрудник желает температуру не выше a i a_i ai​. Гарантируется, что сумма n n n по всем наборам входных данных не превосходит 1 0 3 10^3 103.

Выходные данные

Для каждого набора входных данных выведите n n n строк, i i i-я из которых содержит температуру, удовлетворяющую всех сотрудников с номерами от 1 1 1 до i i i включительно. Если такой температуры не существует, выведите − 1 -1 −1. После вывода ответа на очередной набор входных данных выводите пустую строку. Если ответов несколько, выведите любой.
